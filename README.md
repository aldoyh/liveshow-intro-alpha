# liveshow-intro-alpha

A cinematic splash screen featuring a physics-based plumb bob animation with dramatic timing and professional visual effects.

## Quick Start

Open `index.html` in any modern web browser or serve it with a local HTTP server:

```bash
python3 -m http.server 8080
# Then navigate to http://localhost:8080/index.html
```
